Pubblica la tua app

spritz non ospita i tuoi binari: punta sempre alla tua release originale. Questa pagina ti aiuta a creare il file cicheto (un piccolo YAML) da mettere nel tuo bacaro git. Da li' spritz lo indicizza.

Come funziona. Un bacaro e' un repo git con i tuoi file .yaml. Compila il form, scarica il cicheto, mettilo nel tuo bacaro e comunicane l'URL. Se la tua app e' anche su HaikuPorts, indicalo: spritz manda gli utenti alla versione curata. Siamo additivi a HaikuPorts, non un'alternativa.

1. Accedi

Se hai fatto login, il token e' gia' pronto qui sotto. Serve solo a sapere chi crea il cicheto: il file resta tuo.

2. Dati dell'app

Un link alla tua icona. Meglio se la tieni dove gia' la pubblichi; in alternativa puoi caricarla qui sotto.
