Amministrazione

Gestisci i bacari del registro: ingest, ri-crawl, e rebuild dei repository HaikuDepot. Ogni azione richiede il token admin.

Token admin

Se hai fatto login con un account admin (il primo utente registrato lo e'), questo campo non serve: le azioni usano il tuo accesso. In alternativa incolla il token: e' il valore di SPRITZ_ADMIN_TOKEN impostato all'avvio del server (la chiave operativa del registro, non un account; in sviluppo es. devtoken).
